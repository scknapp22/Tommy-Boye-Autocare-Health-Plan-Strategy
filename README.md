# Health Plan Strategy Analysis for Tommy Boye Autocare

# Objective 
The objective of this project was to develop a cost-effective employer health plan strategy for Tommy Boye Autocare by analyzing plan design, pharmacy benefits, demographics, and projected healthcare claims. The analysis focused on determining whether transitioning to a self-funded health plan would reduce employer costs while maintaining affordable coverage for employees.

Using actuarial modeling and cost projections, we evaluated plan design options, pharmacy benefit manager (PBM) contracts, provider networks, and contribution strategies. The goal was to recommend a health plan structure that balances cost efficiency, employee affordability, and long-term financial sustainability.

# Steps 
Step 1: Plan Design Analysis
The first step was designing plan options that would align with employees’ existing health coverage preferences.
Two plan options were developed:
- Option 1 – High Deductible Health Plan (HDHP)
  - Higher deductible
  - Coinsurance-based cost sharing
  - Compatible with Health Savings Accounts (HSA)
- Option 2 – Base/PPO Plan
  - Lower deductible
  - More predictable copays
  - Richer benefits structure

These options were modeled using the Actuarial Value (ARV) of the current plans.
- HDHP ARV ≈ 69.13%
- Base/PPO ARV ≈ 89.08%

Lower ARV plans generally lead to lower expected employer costs, making Option 1 the more cost-efficient plan.

Step 2: Tier Migration Analysis
To estimate enrollment in the new plan options, a tier migration model was created. Key assumption: Employees would retain their current coverage tier, meaning their dependent coverage would not change.

Example tiers:
- Employee Only
- Employee + Spouse
- Employee + Children
- Employee + Family

Under the migration model:
- Employees currently in HDHP plans moved to Option 1
- Employees in Base/PPO plans moved to Option 2

Projected enrollment:
- Option 1: 280 members
- Option 2: 221 members

Step 3: Pharmacy Benefits Analysis
The next phase evaluated Pharmacy Benefit Manager (PBM) renewal options.
Two formulary structures were compared:
- Standard formulary
- Narrow formulary

Cost analysis included:
- Ingredient costs
- Dispensing fees
- Manufacturer rebates
  
Results:
| Plan Type          | Net Employer Cost |
| ------------------ | ----------------- |
| Current Plan       | $559,769          |
| Standard Formulary | $467,667          |
| Narrow Formulary   | $423,160          |

The narrow formulary option produced the lowest employer cost, making it the recommended pharmacy strategy.

Step 4: Employee Contribution Strategy
Employee contributions were modeled to ensure affordability while controlling employer costs.
Monthly contribution examples:
- Option 1 (HDHP)
  - Employee Only: $125
  - Employee + Spouse: $275
  - Employee + Children: $210
  - Employee + Family: $390

- Option 2 (Base Plan)
  - Employee Only: $160
  - Employee + Spouse: $350
  - Employee + Children: $275
  - Employee + Family: $505
This structure maintained affordability while staying within employer contribution limits.

Step 5: Demographic Risk Analysis
Employee demographic data was analyzed to evaluate overall healthcare risk. Key findings:

Tommy’s Autocare workforce
- Smaller population
- Older employees
- More dependent coverage
- Higher expected claims risk

Boye & Associates workforce
- Larger population
- Younger employees
- More employee-only coverage
- Lower expected claims risk
Combining the two employee groups was expected to lower the overall risk profile of the plan, improving claims performance.

Step 6: Network Discount Analysis
Two provider networks were evaluated based on negotiated discounts and claim costs.
| Network   | Combined Effective Discount | Claim Relativity |
| --------- | --------------------------- | ---------------- |
| Network 1 | **48.8%**                   | 96.6%            |
| Network 2 | 47.5%                       | 99.0%            |

Network 1 produced greater cost savings and lower expected claims, making it the recommended provider network. 

Step 7: Claims Projection
Projected healthcare claims were estimated for the 2026 policy year. Key assumptions:
- Average enrollment: 913 members
- Pharmacy rebates: ≈ $285,735
- One high-cost claimant (“lasered claimant”) adding $400,000 in costs

Projected total annual claims:
| Category         | Cost           |
| ---------------- | -------------- |
| Medical Claims   | $5,436,480     |
| Pharmacy Claims  | $1,197,631     |
| **Total Claims** | **$6,634,111** |

These projections were used to determine premium equivalent rates and plan affordability.

Results:
1. Transition to a self-funded health plan
    - Self-funding allows the employer to capture savings from lower-than-expected claims while maintaining flexibility in plan design.
2. Implement two plan options
    - Option 1: HDHP-style plan (lower cost)
    - Option 2: Base/PPO-style plan (richer benefits)
3. Adopt a narrow pharmacy formulary
    - This option generated the largest cost savings through higher rebates.
4. Select Network 1
    - Network 1 offered higher provider discounts and lower claim relativity.
